## ğŸ“– Tentang Proyek

Wishlist App adalah aplikasi web sederhana yang memungkinkan pengguna untuk mencatat, menyimpan, dan mengelola daftar barang impian mereka. 

Tujuan utama pengembangan proyek ini adalah:
1.  Mendemonstrasikan logika **CRUD** (*Create, Read, Update, Delete*) pada sisi klien.
2.  Mengimplementasikan penyimpanan data menggunakan **LocalStorage** browser (data tidak hilang saat di-refresh).
3.  Melatih logika pemrograman JavaScript dasar tanpa framework.

## âœ¨ Fitur Utama

Berdasarkan implementasi saat ini, fitur yang tersedia meliputi:
* **Create (Tambah):** Input barang baru ke dalam daftar belanja.
* **Read (Lihat):** Menampilkan seluruh daftar barang secara dinamis.
* **Update (Edit):** Mengubah nama barang yang sudah ada melalui fitur edit (*prompt*).
* **Delete (Hapus):** Menghapus barang dari daftar dengan konfirmasi keamanan.
* **Data Persistence:** Data tersimpan otomatis di browser, sehingga aman meskipun tab ditutup.
